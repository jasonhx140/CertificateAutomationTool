====================================================
合格证自动化系统 - Windows 打包说明
====================================================

【方案一】使用GitHub Actions自动打包（推荐）
----------------------------------------------------
1. 在GitHub上创建新仓库
2. 将以下文件提交到仓库:
   - 合格证自动化系统.py
   - 合格证自动化系统.spec
   - version_info.txt
   - requirements.txt
   - .github/workflows/build.yml
3. 推送代码后，在GitHub页面点击 "Actions" 标签
4. 等待构建完成，下载构建产物

优点: 无需Windows电脑，自动化构建


【方案二】在Windows电脑上手动打包
----------------------------------------------------
1. 准备一台Windows电脑（或虚拟机）
2. 安装Python 3.10+ from python.org
3. 双击运行: build_exe.bat
4. 等待打包完成，在 dist/ 目录找到exe文件


【降低杀毒软件误报的方法】
----------------------------------------------------
1. ✅ 本配置已使用 --noupx 参数（禁用UPX压缩）
2. ✅ 添加了详细的版本信息
3. ✅ 使用虚拟环境纯净打包

如仍有误报，客户可以:
- Windows Defender: 点击"允许应用"
- 其他杀毒软件: 添加到白名单


【代码签名证书（可选）】
----------------------------------------------------
如果您有代码签名证书(.pfx文件)，可以签名exe:

signtool sign /f certificate.pfx /p 密码 /t http://timestamp.digicert.com dist\合格证自动化系统.exe

签名后的exe几乎不会被杀毒软件误报。


【测试建议】
----------------------------------------------------
1. 在纯净Windows系统上测试exe运行
2. 上传到 https://www.virustotal.com 检测误报率
3. 如误报率>30%，考虑获取代码签名证书
